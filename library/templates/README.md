# Skeleton-code templates

Read-only starting points stamped by the editor's **File ▸ New Game…**
(docs/game-projects.md §7–8). Each subfolder is one template:

- `empty/` — a bare `GameModule` (init/frame/fixed/draw + `Eng_Run`).
- `fps/` — a working first-person game (controller + HUD) to reshape; the
  current zombies game is its basis.

Both are populated by Step 4 of the migration (the New/Open Game UI). Until then
this folder is a placeholder so the resolver has a `library/templates/` root to
scan.
